Architecture Overview
Three phases. Four tracks. One system.

Built to give every new commercial hire a structured path from offer acceptance to first deal, regardless of region, manager, or start date. Milestone gates replace guesswork with clear readiness signals.

Days 1–30
Learn the Foundation
Context, product, process
  • Company, market, and competitive landscape
  • Product fundamentals and core use cases
  • Internal tools, CRM setup, and workflow orientation
  • Sales methodology introduction (MEDDIC)
  • Shadow 5 live customer calls with debrief
  • ICP and buyer persona deep dive
  • Async eLearning: 8 assigned modules in LMS
Gate: Product assessment pass (80%+), mock discovery call with manager sign-off
Days 31–60
Apply the Skills
Reps, feedback, refinement
  • First solo discovery calls with structured deal review
  • Objection handling lab — 3 live cohort sessions
  • Pipeline building and outbound sequencing
  • Demo certification with senior rep or manager
  • Cross-functional intros: CS, PMM, Solutions
  • Weekly coaching cadence — recorded and logged
  • Async: proposal writing, pricing, and negotiation modules
Gate: 5 qualified opps in Salesforce pipeline, demo certification complete
Days 61–90
Own the Motion
Independence, first results
  • Full pipeline ownership, no shadow required
  • First closed deal or documented path to close
  • Peer teach: present one skill or insight to the team
  • 90-day self-assessment submitted in LMS
  • Manager readiness review and sign-off
  • Feedback loop submitted to L&D for program iteration
  • Transition to standard performance cadence
Gate: Closed deal or advanced-stage opp, manager readiness confirmation

System Architecture
From offer acceptance to day-one assignments — automated.

The program runs on a trigger chain that starts in the HRIS and ends in the LMS before the hire logs in for the first time. No manual enrollment, no missed assignments.

1
Offer accepted — status updated in HRIS
Hire record created with Job Title, Department, Region, Start Date, and Manager fields populated. Triggers downstream workflow.
Workday / BambooHR
2
Role mapped to learning track
Job Title field triggers track assignment logic: AE → Full Ramp track, SDR → Pipeline track, AM → Expansion track, SE → Technical track.
HRIS Workflow / Zapier
3
LMS user account provisioned
User created in LMS with role, region, manager, and cohort group auto-populated. Welcome email queued for 7am on start date. No manual setup required.
LMS Admin
4
Day-one learning path assigned automatically
Track-specific modules assigned with due dates sequenced from start date. Phase 1 modules unlock immediately. Phase 2 unlocks after gate 1 completion. Phase 3 after gate 2.
LMS Rules Engine
5
Manager notified with onboarding dashboard link
Automated email to hiring manager with hire name, track assigned, day-30 gate date, and link to manager-facing progress dashboard. No calendar invite required.
LMS + Email Automation
6
Salesforce contact and onboarding record created
New hire record synced to Salesforce with onboarding status field set to "In Ramp." Pipeline activity, call logs, and deal creation tracked against ramp timeline from day one.
Salesforce + HRIS Sync

Kirkpatrick Framework — All Four Levels
Completion rates are the floor, not the ceiling.

Every phase of The Ramp is tied to a measurable outcome at L1 through L4. The program is iterated quarterly based on gate pass rates, time-to-first-deal data, and cohort comparisons.

Level 1 — Reaction
Did they find it useful?

Post-phase surveys (3-question pulse) sent automatically after each gate. Questions target content relevance, pacing, and manager alignment. Results visible in L&D dashboard within 24 hours.

Baseline result: 4.3/5 avg across Phase 1. Identified demo module pacing as top friction point — rebuilt in Q2.
Level 2 — Learning
Did they retain it?

Knowledge checks embedded in modules (80% pass threshold). Gate assessments at day 28 and day 58. Discovery call and demo certifications evaluated by manager against rubric with standardized scoring.

Baseline result: 84% avg quiz score across cohort. Demo cert pass rate improved from 71% to 89% after adding practice scenario in week 7.
Level 3 — Behavior
Did they do it differently?

Salesforce activity data (call count, pipeline creation, stage progression) benchmarked against pre-program cohort baseline. Manager observation checklist at days 45 and 75. Call recording review with rubric scoring.

Baseline result: Reps in structured ramp averaged 40% more discovery calls in first 60 days vs. unstructured cohort.
Level 4 — Results
Did the business move?

Time-to-first-deal pulled from Salesforce, compared against 12-month pre-program baseline. Pipeline value at day 60 tracked by cohort. Win rate at 6 months compared to historical average. Quarterly L&D review with Sales leadership.

Baseline result: 35% reduction in time-to-first-deal. Average days from start to first close dropped from 118 to 77 after two full cohorts.
